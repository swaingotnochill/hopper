# hopper
Hopper is a smart bookmarking tool to your favourite websites.

## Commands

##### Github:
- gh
- gh @username
- gh search-query
  

##### Stackoverflow:
- st
- st search-query


##### Twitter:
- tw
- tw @username
- tw search-query
  

##### Reddit:
- rd
- rd r/subredditname
- rd search-query
 

##### Linkedin:
- ln
- ln @username
- ln search-query
  
##### To use:
  Add this as your custom search engine in your browser:
  https://fierce-stream-22247.herokuapp.com/
